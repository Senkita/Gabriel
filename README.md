# Gabriel

## Introduction

- ffmpeg 提取视频音轨
- insanely-fast-whisper 语音转文本
- cohere 修正转录稿
- groq 输出总结稿

## Installation

> 循`.env.example`新建`.env`，配置系统变量

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
